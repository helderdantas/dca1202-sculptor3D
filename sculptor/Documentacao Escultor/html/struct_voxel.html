Descrição detalhada

O tipo struct Voxel: Voxels são como pixels, mas utilizados em imagens 3d (possuem 3 dimensões). Os voxels tem como parametros suas informações relativas a cor (red, green e blue) e transparência, além de um boleano que define se o voxel eh visivel ou nao.

Parâmetros
r: float que define a intensidade de vermelho, podendo estar entre [0,1]
g: float que define a intensidade de verde, podendo estar entre [0,1]
b: float que define a intensidade de azul, podendo estar entre [0,1]
a: canal alfa, que define a transparencia do voxel, podendo estar entre [0,1]
isOn: define se o voxel esta ativo ou nao dentro da escultura, sendo 0 par desativado ou 1 para ativado
